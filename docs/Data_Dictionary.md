# Data Dictionary

| Column Name | Description | Data Type | Used In |
| :--- | :--- | :--- | :--- |
| `order_id` | Unique identifier for each individual order/shipment. | Integer / String | Primary Key / Total Shipments Count |
| `order_date` | The date the order was placed and entered the supply chain. | Date | Monthly Trend Time-Series |
| `shipping_mode` | The requested speed of transport (e.g., Same Day, Standard Class). | Text | Emission by Mode KPI |
| `order_region` | The broader geographic destination of the shipment. | Text | Regional Hotspot Analysis |
| `order_city` | The specific destination city for last-mile delivery. | Text | Top 10 Cities Filter |
| `sales` | Total revenue generated by the specific order. | Decimal | Total Sales KPI |
| `quantity` | Number of physical units included in the shipment. | Integer | Volume scaling |
| `estimated_emission_score`| The calculated environmental impact score for the shipment. | Decimal | Core ESG Calculations (Averages) |
